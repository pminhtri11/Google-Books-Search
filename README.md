# React-Google Book Search
Google Book Search App with Save and Delete Function

# Techologies Usage
1.  React
2.  ReactDom
3.  NodeJs
4.  Express
5.  MongoDB

# Start Up
    Open Console
1.  Initialize a package.json: npm init-y
2.  npm install React
3.  npm install ReactDom
4.  npm install axios
5.  npm install express
6.  npm install mongoose


# Deployed Site:
